### 📅 `schedule-app/`

A scheduling management application built around a real teaching environment.

The application provides tools for managing teachers, students, lessons, schedules, and administrative workflows.

```text
schedule-app/
├── Role-based teacher and administrator access
├── Secure password hashing with Argon2
├── Student and teacher management
├── Lesson scheduling
├── Scheduling conflict detection
├── Buffer-time validation
├── Daily schedule dashboards
├── Lesson and student notes
└── SQLite database integration
```

**Technology:** Python, Flask, SQLite, HTML, CSS, JavaScript, Jinja2
